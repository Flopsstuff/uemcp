# Unreal Agent OpenCode Studio Kit

This packaged resource documents the Studio Kit generated by the Unreal Agent editor plugin into each Unreal project:

- `.opencode/agents/` for the primary Unreal Agent and specialist studio roles.
- `.opencode/skills/` for bootstrap, MCP tool selection, prototype, validation, release, and debug loops.
- `.opencode/commands/` for repeatable Unreal production commands.
- `.opencode/plugins/unreal-agent-guardrails.ts` for local OpenCode guardrail hooks.
- `Saved/UnrealAgent/` for workflow state, decisions, and validation evidence.

Runtime generation is implemented in `FUnrealAgentStudioKit`; this resource is shipped so packaged plugin builds have an explicit Studio Kit artifact.
